# リール動画 説明文(案)

Make Life More Comfortable🏡
子育て・家計・家事のこまごまをサポートする無料Webツールをまとめました🌿

📊 ライフプランシミュレーター → 収入・支出・資産運用から将来の資産推移を試算。教育費や税制、運用リターンなど詳細な条件設定にも対応
💸 生活費按分ツール → 夫婦それぞれの手取りに応じて生活費・住宅ローン・教育費の負担割合を自動計算し、可視化
🧹 夫婦の家事分担トラッカー → 家事にポイントを設定し、日々の分担をグラフで見える化
💰 家計管理アプリ → 複数口座を管理、レシート撮影でOCRが自動入力、月々の収支もひと目で確認
👶 出産手続きナビ → 出産予定日を入れるだけで必要な手続きの期限を自動計算、リスト&カレンダーで管理

登録不要・すべて無料で使えます✨
気になるツールがあればコメントで教えてください😊
